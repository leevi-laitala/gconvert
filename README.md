# gconvert

Ghetto general purpose unit converter application.



## Units supported:

```
Units: | Temperature:  | Velocity:          | Volume:              | Mass:         | Lenght:         |
       +–––––––––––––––+––––––––––––––––––––+––––––––––––----------+–––––––––––––––+–––––––––––––––––+
       | <k> Kelvin    | <mph> Miles/h      | <gal> US Gallon      | <lb> Pound    | <in> Inch       |
       | <c> Celsius   | <kmh> Kilometers/h | <l>   Litre          | <kg> Kilogram | <ft> Foot       |
       | <f> Farenheit | <ms>  Meters/s     | <oz>  US Fluid Ounce | <g>  Gram     | <yd> Yard       |
       |               | <kn>  Knots        |                      | <oz> Ounce    | <mi> Mile       |
       |               | <fts> Feet/s       |                      |               | <km> Kilometer  |
       |               |                    |                      |               | <m>  Meter      |
       |               |                    |                      |               | <mm> Millimeter |
```



## Usage:

```
Syntax: command <from> <to> <value>

Example: command c f 34.2    --> converts 34.2C to farenheit
         command mi km 17.5  --> converts 17.5mi to kilometers
```
